========================================
📝 Naming Convention
========================================

Web Routes (HTML):
  - /partial/<feature>         # Fragment
  - /<feature>                 # Full page
  - /<feature>/<id>            # Detail page

API Routes (JSON):
  - /api/<resource>/data       # List
  - /api/<resource>/<id>       # Detail
  - /api/<resource>            # CRUD operations

Classwork Routes:
  - /classwork/lessons/<id>/*  # Classwork specific
  - /classwork/tasks/<id>      # Task operations
  - /classwork/materials/<id>  # Material operations
